POStore - Manual del Sistema Punto de Venta

Formularios Principales

Principal

Propósito: Ventana principal y centro de navegación

  • Proporciona acceso a todos los módulos del sistema
  • Muestra información del negocio y logo
  • Gestiona autenticación y permisos de usuarios
  • Contiene sistema de menús para navegación
  • Muestra panel con métricas clave

Panel de Control

Propósito: Vista general del sistema e indicadores clave

  • Muestra resumen y estadísticas de ventas
  • Muestra alertas de inventario y advertencias de stock bajo
  • Proporciona acceso rápido a tareas comunes
  • Métricas del negocio en tiempo real

Inicio de Sesión

Propósito: Autenticación de usuarios para licencias pagadas

  • Valida credenciales de usuario
  • Determina permisos de usuario (Admin/Regular)
  • Controla acceso a funciones del sistema

Acerca de

Propósito: Información de la aplicación y detalles de versión

  • Muestra versión y detalles de compilación
  • Muestra información de licencia
  • Detalles de contacto y soporte

Formularios de Mantenimiento

Mantenimiento General

Mantenimiento de Usuarios

Propósito: Gestionar usuarios del sistema

  • Agregar, editar y eliminar cuentas de usuario
  • Establecer permisos de usuario (Admin/Regular)
  • Gestionar información de contacto
  • Controlar estado activo de usuarios

Mantenimiento de Configuración

Propósito: Configurar ajustes del sistema

  • Configuración de información del negocio
  • Configuración de recibos (tamaño, impresión)
  • Ajustes de correo para notificaciones
  • Configuración de logo y marca

Mantenimiento de Módulos

Propósito: Gestionar módulos del sistema

  • Configurar ajustes de módulos
  • Establecer numeración consecutiva
  • Asignar tipos de transacción a módulos

Mantenimiento de Tipos de Transacción

Propósito: Gestionar tipos de transacción

  • Definir categorías de transacción
  • Establecer indicadores positivos/negativos
  • Configurar códigos de transacción

Mantenimiento de Inventario

Mantenimiento de Productos

Propósito: Gestionar catálogo de productos

  • Agregar, editar y eliminar productos
  • Establecer precios y costos
  • Configurar niveles mínimos de stock
  • Asignar productos a categorías
  • Vincular productos con proveedores

Mantenimiento de Categorías

Propósito: Organizar productos en categorías

  • Crear y gestionar categorías de productos
  • Organizar estructura de inventario
  • Habilitar reportes por categoría

Mantenimiento de Unidades de Medida

Propósito: Definir unidades de medición

  • Crear unidades de medida (piezas, kg, litros, etc.)
  • Estandarizar medidas de productos
  • Soportar cálculos de inventario

Mantenimiento de Ventas

Mantenimiento de Clientes

Propósito: Gestionar información de clientes

  • Agregar y editar detalles de clientes
  • Establecer límites de crédito y términos de pago
  • Seguimiento de saldos de clientes
  • Configurar tasas de interés para crédito

Mantenimiento de Proveedores

Mantenimiento de Proveedores

Propósito: Gestionar información de proveedores

  • Agregar y editar detalles de proveedores
  • Seguimiento de información de contacto
  • Gestionar relaciones con proveedores

Mantenimiento de Crédito

Mantenimiento de Planes de Pago

Propósito: Configurar planes de pago a crédito

  • Crear planes de pago en cuotas
  • Establecer frecuencias de pago
  • Definir número de cuotas

Formularios de Transacciones

Transacciones de Venta

Ventas

Propósito: Procesar transacciones de venta

  • Crear nuevas facturas de venta
  • Agregar productos a ventas
  • Aplicar descuentos e impuestos
  • Procesar pagos (efectivo/crédito)
  • Imprimir recibos
  • Retener facturas para completar después

Reversión de Ventas

Propósito: Revertir ventas completadas

  • Cancelar o devolver transacciones de venta
  • Restaurar niveles de inventario
  • Procesar reembolsos

Pagos

Propósito: Procesar pagos de clientes

  • Registrar pagos de clientes
  • Aplicar pagos a saldos pendientes
  • Manejar pagos parciales

Facturas en Espera

Propósito: Gestionar transacciones en espera

  • Ver facturas en espera
  • Reanudar transacciones en espera
  • Eliminar facturas abandonadas

Lista de Ventas a Crédito

Propósito: Gestionar ventas a crédito

  • Ver todas las transacciones a crédito
  • Seguimiento de estado de pagos
  • Procesar pagos de crédito

Selección de Plan de Pago

Propósito: Elegir planes de pago para ventas a crédito

  • Seleccionar plan de pago apropiado
  • Configurar términos de cuotas
  • Establecer calendarios de pago

Mostrar Cambio

Propósito: Mostrar cálculo de cambio

  • Mostrar cambio a devolver al cliente
  • Pantalla grande para fácil lectura
  • Confirmación de pago

Transacciones de Inventario

Stock de Inventario

Propósito: Gestionar niveles de stock

  • Ajustar cantidades de stock
  • Registrar movimientos de stock
  • Manejar correcciones de stock
  • Seguimiento de cambios de inventario

Importación de Productos

Propósito: Importar productos desde archivos externos

  • Importar productos desde archivos CSV
  • Creación masiva de productos
  • Validación de datos y manejo de errores

Transacciones Generales

Respaldo de Base de Datos

Propósito: Respaldo y restauración de base de datos

  • Crear respaldos de base de datos
  • Restaurar desde archivos de respaldo
  • Protección segura de datos

Limpieza de Base de Datos

Propósito: Limpiar y optimizar base de datos

  • Eliminar datos antiguos de transacciones
  • Optimizar rendimiento de base de datos
  • Limpiar datos temporales

Ejecutar Scripts

Propósito: Ejecutar scripts SQL

  • Ejecutar comandos SQL personalizados
  • Scripts de mantenimiento de base de datos
  • Operaciones avanzadas de base de datos

Correo Electrónico de Cliente

Propósito: Enviar correos a clientes

  • Enviar recibos por correo
  • Comunicación con clientes
  • Notificaciones automáticas por correo

Reportes

Reporte de Facturas de Venta

Propósito: Reportes de transacciones de venta

  • Ver historial de ventas por rango de fechas
  • Filtrar por cliente, producto o categoría
  • Exportar a formato CSV
  • Análisis de rendimiento de ventas

Reporte de Stock de Inventario

Propósito: Estado actual de inventario

  • Ver niveles actuales de stock
  • Identificar productos con bajo stock
  • Filtrar por categoría o proveedor
  • Reportes de valoración de stock

Reporte de Transacciones de Inventario

Propósito: Historial de movimientos de inventario

  • Seguimiento de todas las transacciones de inventario
  • Ver ajustes y movimientos de stock
  • Filtrar por fecha, producto o tipo de transacción
  • Rastro de auditoría para cambios de inventario

Reporte de Movimientos de Inventario

Propósito: Movimientos detallados de inventario

  • Seguimiento completo de movimientos
  • Análisis de entradas/salidas de inventario
  • Patrones y tendencias de movimiento

Reporte de Productos Más Vendidos

Propósito: Análisis de productos más vendidos

  • Identificar productos más vendidos
  • Análisis de cantidad y ingresos por ventas
  • Filtrar por categoría y proveedor
  • Ranking de rendimiento

Reporte de Transacciones a Crédito

Propósito: Seguimiento de ventas y pagos a crédito

  • Monitorear transacciones a crédito
  • Seguimiento de historial de pagos
  • Reportes de saldos pendientes
  • Análisis de rendimiento de crédito

Reporte de Registros

Propósito: Registros de actividad del sistema

  • Ver historial de actividad del sistema
  • Seguimiento de acciones de usuario
  • Registro de errores y eventos
  • Rastro de auditoría del sistema
Nota: Algunas funciones pueden estar restringidas según su tipo de licencia (Gratuita vs Pagada). Las licencias pagadas proporcionan acceso a reportes avanzados, gestión de usuarios y funciones adicionales.

Consejos de Navegación

  • Use el menú principal para acceder a diferentes módulos
  • Los formularios están organizados por categoría (Mantenimiento, Transacciones, Reportes)
  • La mayoría de los formularios admiten atajos de teclado para navegación rápida
  • Use filtros en reportes para reducir resultados
  • La funcionalidad de exportación está disponible en la mayoría de los formularios de reporte
