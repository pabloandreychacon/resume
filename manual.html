POStore - Point of Sale System Manual

Main Forms

Main

Purpose: Main application window and navigation hub

  • Provides access to all system modules
  • Displays business information and logo
  • Manages user authentication and permissions
  • Contains menu system for navigation
  • Shows dashboard with key metrics

Dashboard

Purpose: System overview and key performance indicators

  • Displays sales summary and statistics
  • Shows inventory alerts and low stock warnings
  • Provides quick access to common tasks
  • Real-time business metrics

Login

Purpose: User authentication for paid licenses

  • Validates user credentials
  • Determines user permissions (Admin/Regular)
  • Controls access to system features

About

Purpose: Application information and version details

  • Shows application version and build information
  • Displays licensing information
  • Contact and support details

Navigation Tips

  • Use the main menu to access different modules
  • Forms are organized by category (Maintenance, Transactions, Reports)
  • Most forms support keyboard shortcuts for faster navigation
  • Use filters in reports to narrow down results
  • Export functionality is available in most report forms
